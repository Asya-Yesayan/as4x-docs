Руководство программиста AS-3X

Данное руководство содержит следующие разделы:

  • Краткое описание платформы AS-3X - архитектура , базовые классы объектов системы, технические требования к среде функционирования
  • Внутренние типы данных - типы, используемые в источниках данных, реквизитах документов и диалоговых окнах
  • Системные описания - приводятся виды системных описаний с примерами и синтаксическим обзором
  • Системные события - события, генерируемые системой при наступлении определенных условий. Объясняется последовательность выполнения обработчиков событий и применимость их к различным описаниям
  • Объекты - описаны объекты, доступные из скрипта, со своими методами и свойствами.
  • Функции - используемые из скрипта и сгруппированные по назначению
  • Системные константы и параметры - используемые из скрипта
  • Структура базы данных - приводится описание баз данных, их структура и назначение
  • Конфигурационный файл - приводится описание конфигурационного файла(config.as)
  • Текстовые импотрные файлы - приводятся описания текстовых импортных файлов системного параметра и дерева-справочника.
