Свойство Parent

См. также  Пример  Применяется к

Возвращает или устанавливает код непосредственного родителя для элемента дерева.

Свойство для чтения и записи.

 

Синтаксис

object.Parent = [sParentKey]

Синтаксис свойства Parent состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент дерева.
sParentKey строковое выражение, указывающее код непосредственного родителя элемента дерева. Заданная пустая строка означает, что элемент дерева является корневым.
