Объект TemplateSubstitutionGrid

Объект TemplateSubstitutionGrid предназначен для описания табличных подстановок в шаблоны печати.

Свойства Описание
FormatedValue

Возвращает символьное значение элемента табличной подстановки.
Name Возвращает идентификатор табличной подстановки.
RowCount

Возвращает количество сторок в табличной подстановке.
Style

Возвращает или устанавливает формат для ячеек таблицы шаблона печати типа .xlsx.
UseStyles

Возвращает признак использования описанных форматов для ячеек таблицы шаблона печати типа .xlsx.
Value Возвращает или устанавливает значение элемента табличной подстановки.
Методы Описание
AddColumn Добавляет колонку в  табличную подстановку.
AddColumnsFromDocumentGrid Добавляет в табличную подстановку колонки из грид-таблицы документа.
AddRow

Добавляет строку в табличную подстановку.
CopyFrom

Копирует табличную подстановку.
FillValuesFromDocumentGrid

Заполняет табличную подстановку из грид-таблицы документа.
SetColumnStyle

Задает форматы ячеек столбца таблицы шаблона печати типа .xlsx.
SetRowStyle

Задает форматы ячеек строки таблицы шаблона печати типа .xlsx.
