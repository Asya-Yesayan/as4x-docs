Объект элемент дерева

Создание элемента дерева выполняется через функцию CreateTreeElement и TreeElProp.

Свойства Описание
Com Возвращает или устанавливает комментарий элемента дерева на родном языке.
DocType Возвращает тип документа для элемента дерева.
ECom Возвращает или устанавливает комментарий элемента дерева на иностранном языке.
ErrorMessage Возвращает или устанавливает текст сообщения об ошибке при попытке добавления в дерево элемента с существующим кодом.
ISN Возвращает внутрисистемный идентификационный код документа элемента дерева.
Key Возвращает или устанавливает ключ (код) элемента дерева.
Leaf Возвращает признак лепестка в дереве.
Parent Возвращает или устанавливает код непосредственного родителя для элемента дерева.
Path Возвращает путь элемента в дереве, начиная с корня дерева.
TreeId Возвращает идентификатор дерева для заданного элемента.
