# Clear մեթոդ

[См. также](../AsGrid.md) Օրինակ [Применяется к](../AsGrid.md) 

Ջնջում է աղյուսակից բոլոր տողերը։

## Շարահյուսություն

``` vb
object.Clear
```


Բաղադրիչներն են՝


| Պարամետր  | Նկարագրություն |
|--|--|
| object | Փաստաթղթի աղյուսակ օբյեկտի հղում։ |
