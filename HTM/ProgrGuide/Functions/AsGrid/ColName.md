# ColName հատկություն

[См. также](../AsGrid.md) Օրինակ [Применяется к](../AsGrid.md) 

Վերադարձնում է փաստաթղթի աղյուսակի ընթացիկ սյունակի ներքին անունը։ 

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.ColName
```

Բաղադրիչներն են՝


| Պարամետր  | Նկարագրություն |
|--|--|
| object | Փաստաթղթի աղյուսակ օբյեկտի հղում։ |
