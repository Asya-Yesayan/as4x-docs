# Caption հատկություն

[См. также](../ICurrentView.md) Օրինակ [Применяется к](../ICurrentView.md)

Վերադարձնում է ընացիկ դիտելու ձևի գլխագիրը։

Կարդալու հատկություն։

## Շարահյուսություն

``` vb
object.Caption
```
Բաղադրիչներն են՝
  
| Պարամետր | Նկարագրություն |
|--|--|
| object | Ընթացիկ դիտելու ձևի հղում։  |
