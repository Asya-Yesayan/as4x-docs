# AllowEdit հատկություն
 
[См. также](../ICurrentTree.md) Օրինակ [Применяется к](../ICurrentTree.md)
  
Վերադարձնում է ընթացիկ ծառում փաստաթղթի խմբագրման թույլտվություն։
 
Կարդալու հատկություն։


## Շարահյուսություն

``` vb
object.AllowEdit
```

Բաղադրիչներն են՝

  
| Параметр | Описание |
|--|--|
| object |  Ընթացիկ ծառի հղում։ |
