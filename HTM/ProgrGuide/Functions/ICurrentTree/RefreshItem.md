# RefreshItem մեթոդ

[См. также](../ICurrentTree.md) Օրինակ [Применяется к](../ICurrentTree.md)
 

Սկսելով նշված հանգույց, բազայից թարմացնում է ծառի ճյուղերը։


## Շարահյուսություն

``` vb
object.RefreshItem()
```
Բաղադրիչներն են՝

  
| Պարամետր | Նկարագրություն |
|--|--|
| object |  Ընթացիկ ծառի հղում։|
