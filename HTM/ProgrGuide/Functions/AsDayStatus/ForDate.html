Свойство ForDate

См. также  Пример  Применяется к

Возвращает или устанавливает дату для определения статуса этой даты.

Свойство для чтения и записи.

 

Синтаксис

object.ForDate[=dDate]  

Синтаксис свойства ForDate состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр статуса дня
dDate выражение типа дата, определяющее дату
