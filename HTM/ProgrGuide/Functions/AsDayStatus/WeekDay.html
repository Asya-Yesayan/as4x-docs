Свойство WeekDay

См. также  Пример  Применяется к

Возвращает день недели для даты.

Свойство для чтения.

 

Синтаксис

object.WeekDay

Синтаксис свойства WeekDay состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр статуса дня

 

Установки

Установки для возвращаемого значения следующие:

Значение Описание
1 воскресенье
2 понедельник
3 вторник
4 среда
5 четверг
6 пятница
7 суббота

 

Тип данных

Целое число
