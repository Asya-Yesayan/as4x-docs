Свойство Comment

См. также  Пример  Применяется к

Возвращает или устанавливает комментарий для файлового дополнения.

Свойство для чтения и записи.

 

Синтаксис

object.Comment [=sValue]

Синтаксис свойства Comment состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта файловое дополнение.
sValue строковое выражение, определяющее комментарий.

 

Тип данных

Строка
