Метод GetLatestVersion

См. также  Пример  Применяется к

Загружает новую версию файлового дополнения с сервера. 

 

Синтаксис

object.GetLatestVersion

Синтаксис метода GetLatestVersion состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта файловое дополнение.
