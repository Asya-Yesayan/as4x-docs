Свойство Parameters

См. также  Пример  Применяется к

Возвращает или устанавливает значения параметров для подстановки в SQL запрос источника данных.

Свойство для чтения и записи.

 

Синтаксис

object.Parameters( index ) [=sValue] 

Синтаксис свойства Parameters состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
index численное выражение, определяющее индекс параметра. Начальное значение индекса 1.
sValue строковое выражение, определяющее значение параметра для подстановки в SQL запрос
