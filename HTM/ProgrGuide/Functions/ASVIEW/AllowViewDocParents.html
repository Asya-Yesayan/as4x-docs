Свойство AllowViewDocParents

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на просмотр родительских документов из вида просмотра.

Свойство для чтения и записи.

 

Синтаксис

object.AllowViewParents[ = bValue ]

Синтаксис свойства AllowViewParents состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
bValue логическое выражение, разрешающее просмотр родительских документов.
