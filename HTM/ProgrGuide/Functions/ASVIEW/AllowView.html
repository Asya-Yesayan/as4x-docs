Свойство AllowView

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на просмотр документа из вида просмотра.

Свойство для чтения и записи.

 

Синтаксис

object.AllowView [=bValue] 

Синтаксис свойства AllowView состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
bValue логическое выражение, разрешающее просмотр документа

 

Установки

Установки для bValue следующие:

Значение Описание
False Не разрешать просмотр документа
True Разрешать просмотр документа
