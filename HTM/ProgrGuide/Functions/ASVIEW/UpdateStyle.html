Свойство UpdateStyle

См. также  Пример  Применяется к

Возвращает или устанавливает стиль обновления при редактировании документа из вида просмотра.

Свойство для чтения и записи.

 

Синтаксис

object.UpdateStyle [=bValue] 

Синтаксис свойства UpdateStyle состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта вида просмотра
bValue логическое выражение, обозначающее стиль обновления

 

Установки

Установки для bValue следующие:

Значение Описание
False происходит обновление всей папки
True обновление происходит по ISN-у документа
