Свойство Parent

 См. также  Пример  Применяется к

Возвращает код непосредственного родителя выбранного элемента иерархического вспомогательного списка. Если, конечно, осуществлен выбор и значение свойства isSelected=True.

Свойство для чтения.

 

Синтаксис

object.Parent

Синтаксис свойства Parent состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа иерархический вспомогательный список.

 

Тип данных

Строка
