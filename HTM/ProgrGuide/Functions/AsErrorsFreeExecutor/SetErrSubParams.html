Метод SetErrSubParams

 См. также  Пример  Применяется к

Устанавливает параметры для процедуры вывода дополнительных сообщений об ошибке. Если параметры не устанавливаются, то процедура вызывается с параметрами, что были переданы методу Run.

 

Синтаксис

object.SetErrSubParams xParamsArray()

Синтаксис свойства SetErrSubParams состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта исполнителя процедур.
xParamsArray() массив параметров для подстановки в процедуру в качестве фактических параметров.
