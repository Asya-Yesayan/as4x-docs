Свойство OnErrorRollBack

См. также  Пример  Применяется к

Возвращает или устанавливает признак отката транзакции при ошибке. По умолчанию транзакция откатывается (OnErrorRollBack = True).

Свойство для чтения и записи.

 

Синтаксис

object.OnErrorRollBack [=bValue] 

Синтаксис свойства OnErrorRollBack состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта AsErrorsFreeExecutor.
bValue логическое выражение, определяющее откатывать ли транзакцию при возникновении ошибки.
