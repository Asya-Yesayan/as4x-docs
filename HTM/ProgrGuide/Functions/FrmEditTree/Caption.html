Свойство Caption

См. также  Пример  Применяется к

Возвращает значение заголовка текущего дерева.

Свойство для чтения.

Синтаксис

object.Caption

Синтаксис свойства Caption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на форму прокрутки деревьев-справочников

Тип данных

Строковое выражение
