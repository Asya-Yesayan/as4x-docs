Свойство AllowAddNode

См. также  Пример  Применяется к

Возвращает разрешение на добавление узла в текущем дереве.

Свойство для чтения.

Синтаксис

object.AllowAddNode

Синтаксис свойства AllowAddNode состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на форму прокрутки деревьев-справочников

Тип данных

Логический
