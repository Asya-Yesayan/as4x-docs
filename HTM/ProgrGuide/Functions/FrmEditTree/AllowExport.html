Свойство AllowExport

См. также  Пример  Применяется к

Возвращает разрешение на экспортирование документа из текущего дерева.

Свойство для чтения.

Синтаксис

object.AllowExport

Синтаксис свойства AllowExport состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на форму прокрутки деревьев-справочников

Тип данных

Логический
