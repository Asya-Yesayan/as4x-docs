Метод ShowStatus

 См. также  Пример  Применяется к

Выводит сообщение в статусную строку окна приложения. После срабатывания метода ShowStatus происходит автоматический вызов метода Refresh, для перерисовки экрана.

 

Синтаксис

object.ShowStatus (sMessage)

Синтаксис метода ShowStatus состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект типа обновитель экрана
sMessage строковое выражение, определяющее сообщение для показа в статусной строке.
