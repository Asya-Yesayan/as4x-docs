Свойство GLAccISN

См. также  Пример  Применяется к

Возвращает или устанавливает внутрисистемный код ISN аккумулирующего объекта.

Свойство для чтения и записи.

 

Синтаксис

object.GLAccISN = [nISN]

Синтаксис свойства GLAccISN состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
nISN переменная типа Long, определяющая значение внутрисистемного кода ISN аккумулирующего объекта.
