Свойство Info

См. также  Пример  Применяется к

Возвращает или устанавливает значение поля для специальной обработки. Значения этого свойства не запоминется в БД.

Свойство для чтения и записи.

 

Синтаксис

object.Info [=Value]   

Синтаксис свойства Info состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
Value выражение типа Variant.
