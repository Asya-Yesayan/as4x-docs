Свойство Spec

См. также  Пример  Применяется к

Возвращает или устанавливает значение специальной строки для хранения различной дополнительной информации о проводке. Максимально допустимая длина спец-строки 255 символов.

Свойство для чтения и записи.

 

Синтаксис

object.Spec [=sValue]   

Синтаксис свойства Spec состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
sValue строковое выражение, определяющее значение спец-строки.
