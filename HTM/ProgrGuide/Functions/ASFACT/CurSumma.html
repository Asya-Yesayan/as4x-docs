Свойство CurSumma

См. также  Пример  Применяется к

Возвращает или устанавливает для проводки сумму в валюте. После задания суммы, ядро системы автоматически округляет его до заданной степени округления из описания учета.

Свойство для чтения и записи.

 

Синтаксис

object.CurSumma = [cCurSum]

Синтаксис свойства CurSumma состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
cCurSum переменная типа Currency, определяющая валютную сумму проводки.
