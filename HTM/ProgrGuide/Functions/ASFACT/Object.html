Свойство Object

См. также  Пример  Применяется к

Возвращает или устанавливает объект проводки.

Свойство для чтения и записи.

 

Синтаксис

factobject.Object [=sValue]   

Синтаксис свойства Object состоит из следующих частей:

Параметр Описание
factobject переменная, ссылающаяся на объект типа проводка.
sValue строковое выражение, определяющее код объекта проводки.

 

Примечание

Объект проводки должен находится в папке MainFolder.
