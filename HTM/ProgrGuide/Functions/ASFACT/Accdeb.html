Свойство AccDeb

См. также  Пример  Применяется к

Возвращает или устанавливает дебетуемый объект проводки.

Свойство для чтения и записи.

 

Синтаксис

object.AccDeb [=vValue]   

Синтаксис свойства AccDeb состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
vValue выражение типа Variant, определяющее дебетуемый объект, в частности счет.
