Свойство AccCrdISN

См. также  Пример  Применяется к

Возвращает или устанавливает внутрисистемный код ISN для кредитуемого объекта.

Свойство для чтения и записи.

 

Синтаксис

object.AccCrdISN = [nISN]

Синтаксис свойства AccCrdISN состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
nISN переменная типа Long, определяющая значение внутрисистемного кода ISN для кредитуемого объекта.
