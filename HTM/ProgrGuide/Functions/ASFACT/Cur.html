Свойство Cur

См. также  Пример  Применяется к

Возвращает или устанавливает код валюты проводки.

Свойство для чтения и записи.

 

Синтаксис

object.Cur [= nValue]   

Синтаксис свойства Cur состоит из следующих частей:

Параметр Описание
object переменная, ссылающаяся на объект типа проводка.
nValue числовое выражение, определяющее код валюты проводки.
