Свойство BookMark

См. также  Пример  Применяется к

Возвращает уникальный ключ для текущей строки папки. Перемещения позиции указателя текущей строки из скрипта не приводят к изменению указателя выбранной текущей строки на экране. Если значения свойств EOF или BOF являются истинными, то использование данного свойства сгенерирует сообщение об ошибке.

Свойство для чтения.

 

Синтаксис

object.BookMark

Синтаксис свойства BookMark состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.

 

Тип данных

Длинное целое число
