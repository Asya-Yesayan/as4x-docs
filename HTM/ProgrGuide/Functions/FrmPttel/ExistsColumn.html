Метод ExistsColumn

См. также  Пример  Применяется к

Возвращает наличие колонки в текущем виде просмотра.

Синтаксис

object.ExistsColumn(columnName)

Синтаксис метода ExistsColumn состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.
columnName строковое выражение, определяющее идентификатор перманентной колонки в текущем виде просмотра

Тип данных

Логический
