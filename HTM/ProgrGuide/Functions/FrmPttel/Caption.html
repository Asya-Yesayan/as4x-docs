Свойство Caption

См. также  Пример  Применяется к

Возвращает заголовок текущего вида просмотра.

Свойство для чтения.

Синтаксис

object.Caption

Синтаксис свойства Caption состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.

Тип данных

Строковое выражение
