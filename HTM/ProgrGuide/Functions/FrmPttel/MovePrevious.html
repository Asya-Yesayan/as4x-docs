Метод MovePrevious

См. также  Пример  Применяется к

Перемещает указатель записи на предыдущую строчку в текущей папке. При использовании данного метода в виде просмотра с истинным значением свойства BOF генерируется ошибка. Если данный метод используется с первой текущей строчки, то значение свойства BOF становится истинным.

 

Синтаксис

object.MovePrevious

Синтаксис метода MovePrevious состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на текущую папку.
