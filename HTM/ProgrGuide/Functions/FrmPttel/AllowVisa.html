Свойство AllowVisa

См. также  Пример  Применяется к

Возвращает или устанавливает разрешение на визирование документа из текущего вида просмотра.

Свойство для чтения и записи.

 

Синтаксис

object.AllowVisa [=bValue] 

Синтаксис свойства AllowVisa состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта текущий вид просмотра
bValue логическое выражение, разрешающее визирование документа

 

Установки

Установки для bValue следующие:

Значение Описание
False Не разрешать визирование документа
True Разрешать визирование документа
