Функция GetTemplateExtension

Возвращает расширение по типу.

Возвращаемое значение строка.

 

Синтаксис

GetTemplateExtension(sTemplateType)

Синтаксис функции GetTemplateExtension состоит из следующих частей:

Параметр Описание
sTemplateType обязательное строковое выражение, определяющее тип шаблона.

 

Установки

Тип шаблона Расширение по типу
"0" ".doc"
"1" ".xls"
"2" ".docx"
"3" ".xlsm"
"4" ".pdf"
"5" ".odt"
"6" ".ods"
"8" ".xlsx"
"9" ".txt"
