Функция AreValuesEqual

Возвращает значение True, если значение vValue1 совпадает со значением vValue2.

 

Синтаксис

AreValuesEqual(vValue1, vValue2)

Синтаксис функции AreValuesEqual состоит из следующих частей:

Параметр Описание
vValue1 обязательное выражение типа вариант, определяющее первое проверяемое значение.
vValue2 обязательное выражение типа вариант, определяющее второе проверяемое значение.
