Функция CreateRefresher
свойства   методы

Создает новый экземпляр объекта обновителя экрана, для перерисовки экрана и изменения значения статусной строки.

 

Синтаксис

set sNewRefresher = CreateRefresher

Синтаксис функции CreateRefresher состоит из следующих частей:

Параметр Описание
sNewRefresher переменная, ссылающаяся на новый экземпляр обновителя экрана
