Функция IsTemplAvailable

Возвращает доступность шаблона печати для данного документа.

Возвращаемый тип  значения логическое.

 

Синтаксис

IsTemplAvailable(sTemplateName, [sTemplateType],  [xDoc])

Синтаксис функции IsTemplAvailable состоит из следующих частей:

Параметр Описание
sTemplateName обязательное строковое выражение, определяющее имя шаблона.
sTemplateType обязательное строковое выражение, определяющее тип шаблона.
xDoc Документ, для которого проверяется доступность шаблона печати.
