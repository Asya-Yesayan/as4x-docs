Функция SplitString

Возвращает одномерный массив, содержащий подстроки фиксированной ширины. Заданная строка расщепляется на подстроки по заданному разделителю.

 

Синтаксис

SplitString (sExpression, nLen, [sDelimiter])

Синтаксис функции SplitString состоит из следующих частей:

Параметр Описание
sExpression строковое выражение, определяющее исходную строку, которую необходимо разбить на подстроки.
nLen численное выражение, определяющее максимальное количество символов, содержащихся в любом элементе массива (т.е. ширину полученных строк).
sDelimiter необязательное строковое выражение, определяющее символ разделителя в строке. Она определяет границы подстрок. По умолчанию в качестве разделителя принимается значение " " пробел.
