Функция FolderElProp
свойства   методы

Возвращает ссылку на существующий объект типа элемент папки по имени папки и ключу. Функция используется для получения значений свойств элемента папки. Если объект не существует, тогда функция возвращает значение Nothing.

 

Синтаксис

set xFolderEl = FolderElProp(FolderId, Key)

Синтаксис функции FolderElProp состоит из следующих частей:

Параметр Описание
xFolderEl переменная, ссылающаяся на объект типа элемент папки
FolderId строковое выражение, определяющее идентификатор папки
Key строковое выражение, определяющее ключ документа в папке

Пример функции FolderElProp

Переменная xFOLDERACC ссылается на существующий объект типа элемент папки. Через нее возвращаются значения свойств Сom и Spec элемента папки.

set xFOLDERACC = FolderElProp("NBACC", doc("NBACC"))
GetNBNAME = xFOLDERACC.Com
CodCur = MID(xFOLDERACC.Spec, 1, 3)
