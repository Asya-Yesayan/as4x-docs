Метод ShowTemplateFileUsingDefaultApp

Показывает заполненный файл шаблона.

 

Синтаксис

ShowTemplateFileUsingDefaultApp(fileName)

Синтаксис метода ShowTemplateFileUsingDefaultApp состоит из следующих частей:

Параметр Описание
fileName нобязательное строковое выражение, определяющее имя файла шаблона.
