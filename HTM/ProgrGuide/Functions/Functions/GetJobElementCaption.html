Функция GetJobElementCaption

Возвращает заголовок задания из таблицы SYSDEF, без авторизации в системе.

Возвращаемое значение строковое.

 

Синтаксис

GetJobElementCaption(sJobElementName, [iLang])

Синтаксис функции GetJobElementCaption состоит из следующих частей:

Параметр Описание
sJobElementName строковое выражение, определяющее идентификатор задания
iLang численное выражение, определяющее язык возвращаемого заголовка задания. При значении -1 выбирается текущий системный язык, при значении 1- армянский, при остальных значениях - английский.
