Функция AllowCreateDoc

Проверяет доступен ли указанный тип документа для добавления из данного рабочего места. Она возвращает истинное значение, если в навигаторском дереве существует ветвь для создания нового экземпляра документа.

Возвращаемое значение логическое.

 

Синтаксис

AllowCreateDoc (DocType)

Синтаксис функции AllowCreateDoc состоит из следующих частей:

Параметр Описание
DocType строковое выражение, определяющее идентификатор документа из его описания.

 

Примечание

Данная функция может использоваться для определения возможности редактирования, удаления или же другого действия над документом, в зависимости от возможности его добавления из рабочего места.
