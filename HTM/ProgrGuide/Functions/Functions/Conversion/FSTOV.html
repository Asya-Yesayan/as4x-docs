Функция FSTOV

Преобразовывает форматированное внешнее представление в соответствующее внутреннее представление для хранения в базе данных.

Возвращается значение типа Variant.

 

Синтаксис

FSTOV (strType, sValue)

Синтаксис функции FSTOV состоит из следующих частей:

Параметр Описание
strType строковое выражение, определяющее тип данных для преобразования.
sValue строковое выражение, определяющее форматированное исходное значение, подлежащее преобразованию.
