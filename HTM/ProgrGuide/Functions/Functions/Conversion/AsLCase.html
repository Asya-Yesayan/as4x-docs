Функция AsLCase

Изменяет символы заданной строки в нижний регистр (в строчные буквы) и возвращает полученную строку.

Возвращается символьная строка.

 

Синтаксис

AsLCase (sSource, [nLang])

Синтаксис функции AsLCase состоит из следующих частей:

Параметр Описание
sSource строковое выражение, определяющее исходную строку
nLang необязательное численное выражение целого типа, определяющее код языка, символы которого содержит исходная строка.

 

Установки

Установки для Lang следующие:

Значение Описание
-1 значение по умолчанию, определяющее код рабочего языка системы из параметров.
1 Армянский язык (кодировка Windows)
2 Русский язык (кодировка Windows)
3 Английский язык
