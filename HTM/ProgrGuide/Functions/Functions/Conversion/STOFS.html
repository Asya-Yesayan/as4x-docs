Функция STOFS

Преобразовывает строковое значение заданного типа в форматированное строковое представление. Длина результирующей строки равна максимальной длине указанного типа.

Возвращается символьная строка.

 

Синтаксис

STOFS (strType, Value)

Синтаксис функции STOFS состоит из следующих частей:

Параметр Описание
strType строковое выражение, определяющее тип данных для преобразования. Исходя из длины типа определяется длина форматированной строки.
Value строковое выражение, определяющее исходное значение, подлежащее преобразованию.

 

Примечание

В отличие от функции VTOFS, данная функция в качестве аргумента исходного значения имеет строку, а не Variant-у.
