Функция GetJobElementDescription

Возвращает краткое описание задания.

Возвращаемое значение строковое.

 

Синтаксис

GetJobElementDescription(sJobElementName)

Синтаксис функции GetJobElementDescription состоит из следующих частей:

Параметр Описание
sJobElementName строковое выражение, определяющее идентификатор задания
