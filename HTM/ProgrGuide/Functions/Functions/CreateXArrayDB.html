Функция CreateXArrayDB

Создает объект типа XArrayDB и возвращает ссылку на него.

 

Синтаксис

set sNewXArrayDB = CreateXArrayDB([RowsCount],[ColumnsCount])

Синтаксис функции CreateXArrayDB состоит из следующих частей:

Параметр Описание
RowsCount необязательное числовое выражение, определяющее количество строк в создаваемом XArrayDB.Если параметр не задан, то количество строк равно 0.
ColumnsCount необязательное числовое выражение, определяющее количество столбцов в создаваемом XArrayDB. Если параметр не задан, то размерность создаваемого XArrayDB не задана.

 

Примечание

Свойства и методы доступные для объекта xArrayDB смотрите в помощи данного объекта.
