Свойство BackColor

См. также  Пример  Применяется к

Возвращает или устанавливает фоновый цвет ячейки.

 

Синтаксис

object.BackColor[= value]

Синтаксис свойства BackColor состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа TemplateSubstCellStyle.
value строковое выражение, определяющее фоновый цвет ячейки, которое начинается с символа # и представляется в виде трех пар цифер шестнадцатеричной системы цветовой модели  RGB.
