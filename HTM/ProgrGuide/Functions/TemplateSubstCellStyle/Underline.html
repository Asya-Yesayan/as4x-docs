Свойство Underline

См. также  Пример  Применяется к

Возвращает или устанавливает подчеркиваемость шрифта ячейки.

 

Синтаксис

object.Underline[= value]

Синтаксис свойства Underline состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа TemplateSubstCellStyle.
value численное выражение, oпределяющее подчеркиваемость шрифта ячейки. Принимает значения 0, 1 или 2. При значении 1 устанавливается подчеркиваемость шрифта ячейки, при значении 0 подчеркиваемость шрифта снимается, при значении 2 ничего не меняется.  По умолчанию принимает значение 2.
