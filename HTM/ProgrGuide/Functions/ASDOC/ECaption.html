Свойство ECaption

См. также  Пример  Применяется к

Возвращает заголовок-наименование документа на иностранном языке из его описания.

Свойство для чтения.

 

Синтаксис

object.ECaption

Синтаксис свойства ECaption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Строка
