Метод HiParDelete

 См. также  Пример  Применяется к

Удаляет те параметры из таблицы HIPAR, документ-основанием которых является загруженный документ.

 

Синтаксис

object.HiParDelete

Синтаксис метода HiParDelete состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа.
