Свойство CreatorSUID

 См. также  Пример  Применяется к

Возвращает идентификатор создателя документа.

Синтаксис

object.CreatorSUID

Синтаксис свойства CreatorSUID состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
