Метод Confirm

Визирует документ.

См. также Пример Применяется к

Синтаксис

object.Confirm([intCheckLevel], [sLogComment])

Синтаксис свойства Confirm состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа
intCheckLevel необязательное численное выражение целого типа, определяющее уровень проверки документа. По умолчанию принимает значение 0
sLogComment необязательное строковое выражение, определяющее сообщение, которое будет записано в журнале регистрации операций над документом .
