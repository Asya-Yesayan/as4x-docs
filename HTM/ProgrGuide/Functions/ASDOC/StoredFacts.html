Свойство StoredFacts

См. также  Пример  Применяется к

Возвращает коллекцию сохраненных проводок. Данный метод нужно вызывать из обработчика системного события Action.

Свойство только для чтения.

 

Синтаксис

object.StoredFacts

Синтаксис свойства StoredFacts состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Коллекция
