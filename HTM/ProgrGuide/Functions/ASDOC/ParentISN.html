Свойство ParentISN

См. также  Пример  Применяется к

Возвращает внутрисистемный уникальный идентификатор ISN родительского документа. Если родитель отсутствует то возвращается -1.

Свойство для чтения.

 

Синтаксис

object.ParentISN

Синтаксис свойства ParentISN состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Длинное целое
