Свойство NextTrans

См. также  Пример  Применяется к

Возвращает номер очередной транзакции, которая не увеличивается на единицу при каждом его запросе.

Свойство для чтения.

 

Синтаксис

object.NextTrans

Синтаксис свойства NextTrans состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр документа

 

Тип данных

Длинное целое
