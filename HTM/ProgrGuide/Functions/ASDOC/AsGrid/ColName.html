Свойство ColName

См. также  Пример  Применяется к

Возвращает идентификатор текущей колонки в грид таблице документа.

Свойство для чтения.

 

Синтаксис

object.ColName

Синтаксис свойства ColName состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе

 

Тип данных

Строка
