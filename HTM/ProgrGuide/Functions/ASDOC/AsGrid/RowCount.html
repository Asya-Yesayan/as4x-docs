Свойство RowCount

См. также  Пример  Применяется к

Возвращает количество строк в грид-таблице документа.

 

Синтаксис

object.RowCount

Синтаксис свойства RowCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе

 

Тип данных

Длинное целое число
