Свойство GridControl

См. также  Пример  Применяется к

Возвращает объект TrueDBGrid грид таблицы документа.

Свойство для чтения.

 

Синтаксис

object.GridControl

Синтаксис свойства GridControl состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на грид-таблицу в документе

 

Тип данных

Объект
