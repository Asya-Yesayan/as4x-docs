Объект AsQuery

См. также  Пример


Объект AsQuery предназначен для выполнения параметризированного запроса.

Свойства Описание
SQL Возвращает или устанавливает текст SQL-запроса.
Parameters Возвращает или устанавливает значение параметра для SQL-запроса.

 

Методы Описание
ExecuteReader Выполняет SQL-запрос и возвращает результат выполнения.
ExecuteNonQuery Выполняет SQL-запрос.
ExecuteScalar Выполняет SQL-запрос и возвращает первый/единственный элемент из результата выполнения.
