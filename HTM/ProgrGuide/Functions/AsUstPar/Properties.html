Свойство Properties

См. также  Пример  Применяется к

Возвращает ссылку на объект Dictionary. Данное свойство применяется для хранения некоторых значений для последующей их обработки. Доступ и обновление элементов коллекции происходит с помощью методов и свойств объекта Dictionary.

 

Синтаксис

object.Properties

Синтаксис свойства Properties состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа диалог.
