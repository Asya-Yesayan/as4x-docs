Свойство EventModule

См. также  Пример  Применяется к

Возвращает или устанавливает идентификатор модуля, где собраны одноименные процедуры для обработки системных событий диалога.

Свойство для чтения и записи.

 

Синтаксис

object.EventModule = [sValue]

Синтаксис свойства EventModule состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
sValue строковое выражение, определяющее идентификатор модуля.
