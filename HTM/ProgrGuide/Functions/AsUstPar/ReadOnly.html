Свойство ReadOnly

См. также  Пример  Применяется к

Возвращает или устанавливает статус режима редактирования реквизита пользовательского диалога.

Свойство для чтения и записи.

 

Синтаксис

object.ReadOnly(RekvName)[=bValue]  

Синтаксис свойства ReadOnly состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
RekvName строковое выражение, определяющее идентификатор реквизита пользовательского диалога
bValue логическое выражение, определяющее возможность редактирования указанного реквизита

 

Тип данных

Логический
