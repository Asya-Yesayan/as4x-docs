Метод AddLine

См. также  Пример  Применяется к

Добавляет разделительную линию в пользовательский диалог. Она необходима для группировок при большом количестве реквизитов в диалоге.

 

Синтаксис

object.AddLine([Caption],[Alignment], [ECaption])

Синтаксис метода AddLine состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
Caption необязательное строковое выражение, определяющее текст на разделительной линии.
Alignment необязательное численное выражение, определяющее выравнивание текста на разделительнои линии.
При значении 0 определяет выравнивание по середине, 1 - выравнивание на лево, 2 - выравнивание на право. Значение по умолчанию - 0.
ЕCaption необязательное строковое выражение, определяющее текст на разделительной линии на иностранном языке.
