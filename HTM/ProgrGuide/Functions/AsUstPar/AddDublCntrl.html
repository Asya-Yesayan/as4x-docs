Метод AddDublCntrl

См. также  Пример  Применяется к

Добавляет вторичный реквизит ввода в пользовательский диалог. Причем данный реквизит на форме располагается не к низу от текущего реквизита, а рядом с ним, наследуя от него тип и имея общий с ним заголовок.

 

Синтаксис

call object.AddDublCntrl(DublRekvName, RekvName, [AttribRekv], [ValueRekv], [iSaveValue], [sCaption] )

Синтаксис метода AddDublCntrl состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта пользовательского диалога.
DublRekvName строковое выражение, определяющее идентификатор вторичного реквизита в диалоге
RekvName строковое выражение, определяющее идентификатор реквизита в диалоге, рядом с которым будет расположен данный реквизит
AttribRekv необязательное строковое выражение, определяющее атрибуты реквизита (их может быть несколько).
ValueRekv необязательное выражение типа Variant, определяющее первоначальное значение реквизита.
iSaveValue необязательное численное выражение, определяющее признак запоминания значений реквизита. При 1 - текущее значение реквизита запоминается в реестре, а при значении 0 - нет. По умолчанию принимает значение 1.
sCaption необязательное строковое выражение, определяющее заголовок реквизита в диалоге.

 

Примечание

Свой тип вторичный реквизит наследует от реквизита, указанного в RekvName.
