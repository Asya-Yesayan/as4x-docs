Свойство Com

См. также  Пример  Применяется к

Возвращает или устанавливает комментарий элемента папки на родном языке. Комментарий появляется в списке в виде наименования, во время ссылок на справочники типа Folder.

Свойство для чтения и записи.

 

Синтаксис

object.Com = [sValue]

Синтаксис свойства Com состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент папки.
sValue строковое выражение, определяющее значение комментария к элементу папки.
