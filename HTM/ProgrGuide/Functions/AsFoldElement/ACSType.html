Свойство ACSType

См. также  Пример  Применяется к

Возвращает или устанавливает тип доступа элемента папки. Свойство для чтения и записи.

 

Синтаксис

object.ACSType = [sValue]

Синтаксис свойства ACSType состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент папки.
sValue строковое выражение, определяющее тип доступа элемента в папке. Ее максимальная длина может быть 2 символов.
