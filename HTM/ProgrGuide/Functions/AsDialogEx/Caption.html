Свойство Caption

См. также  Пример  Применяется к

Возвращает или устанавливает заголовок описательного диалога.

Свойство для чтения и записи.

 

Синтаксис

object.Caption = [sValue]

Синтаксис свойства Caption состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта описательного диалога.
sValue строковое выражение, определяющее заголовок описательного диалога.
