Свойство ChangeTypeControl

См. также  Пример  Применяется к

Возвращает или устанавливает тип для данного элемента управления диалога. Изменение типа допустима только для Folder() Можно изменить только наименование папки, если длина ключей папок совпадает.

Свойство для чтения и записи.

 

Синтаксис

object.ChangeTypeControl(sControlName) = [sNewType]

Синтаксис свойства ChangeTypeControl состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта описательного диалога.
sControlName строковое выражение, определяющее идентификатор элемента управления в описательном диалоге.
sNewType строковое выражение, определяющее тип элемента управления диалога.
