Свойство ReadOnly

См. также  Пример  Применяется к

Возвращает или устанавливает статус режима редактирования заданного элемента управления.

Свойство для чтения и записи.

 

Синтаксис

object.ReadOnly(sControlName) = [bReadOnly]

Синтаксис свойства ReadOnly состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта описательного диалога.
sControlName строковое выражение, определяющее имя элемента управления в описательном диалоге.
bReadOnly выражение логического типа, определяющее статус редактирования   соответствующего злемента управления. При значении True - возможно только чтение значения, при False - разрешаются и чтение и запись.

 

Тип данных

Логический
