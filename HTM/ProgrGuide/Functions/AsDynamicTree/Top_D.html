Свойство Top

См. также  Пример  Применяется к

Возвращает или устанавливает удаленность верхнего края динамического дерева от верхнего края окна приложения.

Свойство для чтения и записи.

 

Синтаксис

object.Top = [nTopValue]

Синтаксис свойства Top состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа динамическое дерево.
nTopValue выражение типа Single, определяющее расстояние между верхним краем динамического дерева и верхним краем окна приложения.

 

Тип данных

Single
