Свойство Kind

 См. также  Пример  Применяется к

Возвращает тип элемента структуры документа.

Свойство только для чтения.

Значение свойства Kind Тип элемента структуры документа

0

 Реквизит
1 Грид-таблица
2 Изображение
3 Мемо поле
4 Колонка грид-таблицы

 

Синтаксис

object.Kind

Синтаксис свойства Kind состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент структуры документа.

 

Тип данных

Целое число
