Свойство SubElementsCount

 См. также  Пример  Применяется к

Возвращает количество структурных элементов второго уровня (колонок грид-таблицы).

Свойство только для чтения.

 

Синтаксис

object.SubElementsCount

Синтаксис свойства SubElementsCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент структуры документа.

 

Тип данных

Целое число
