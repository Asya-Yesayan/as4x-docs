Свойство Name

 См. также  Пример  Применяется к

Возвращает идентификатор элемента структуры документа.

Свойство только для чтения.

 

Синтаксис

object.Name

Синтаксис свойства Name состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта элемент структуры документа.

 

Тип данных

Строка
