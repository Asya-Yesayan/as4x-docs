Свойство NativeCaption

См. также  Пример  Применяется к

Возвращает или устанавливает заголовок документа на основном языке системы.

Синтаксис

object.NativeCaption

Синтаксис свойства NativeCaption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта структура документа.

 

Тип данных

Строка
