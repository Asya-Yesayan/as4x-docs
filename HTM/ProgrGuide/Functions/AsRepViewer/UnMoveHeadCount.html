Свойство UnMoveHeadCount

См. также  Пример  Применяется к

Возвращает или устанавливает количество неперемещаемых заголовков. При горизонтальной прокрутке фрагментов неперемещаемый заголовок центрируется относительно размеров формы отчета, оставаясь видимым.

Свойство для чтения и записи.

 

Синтаксис

object.UnMoveHeadCount = [nValue]

Синтаксис свойства UnMoveHeadCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
nValue численное выражение целого типа, определяющее количество неперемещаемых заголовков
