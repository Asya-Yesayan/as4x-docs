Свойство Caption

См. также  Пример  Применяется к

Возвращает или устанавливает заголовок отчета (справки).

Свойство для чтения и записи.

 

Синтаксис

object.Caption[ = sValue ]

Синтаксис свойства Caption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
sValue строковое выражение, определяющее заголовок отчета
