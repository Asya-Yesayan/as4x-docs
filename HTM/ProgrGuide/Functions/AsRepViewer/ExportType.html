Свойство ExportType

См. также  Пример  Применяется к

Возвращает или устанавливает тип кодировки для файла экспортируемой справки. Данное свойство применяется для задания кодировки по умолчанию, при экспорте файла, которое можно всегда изменить.

Свойство для чтения и записи.

 

Синтаксис

object.ExportType[ = iValue ]

Синтаксис свойства ExportType состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
iValue числовое выражение, определяющее тип кодировки для текстовых файлов. При 0 - кодировка ANSI для операционной системы Windows. При 1 - кодировка OEM для операционной системы DOS.
