Свойство HasDefinedStyle

См. также  Пример  Применяется к

Возвращает признак задания стиля экспорта строк справки в Excel. Если для данного отчета уже определена стиль, то возвращаемое значение будет равно True, в противнном случае - False.

Свойство для чтения и записи.

 

Синтаксис

object.HasDefinedStyle

Синтаксис свойства HasDefinedStyle состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
