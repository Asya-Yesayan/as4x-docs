Свойство GetRows

См. также  Пример  Применяется к

Возвращает массив строк из справки, начиная с указанной строки с указанным количеством. Размерность массива определяется из количества фрагментов в справке.

 

Синтаксис

object.GetRows (lStartRow, lRowCount)

Синтаксис свойства GetRows состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа отчет.
lStartRow численное выражение, определяющее номер строки, начиная с которой возвращаются строки справки.
lRowCount численное выражение, определяющее количество возвращаемых строк справки.
