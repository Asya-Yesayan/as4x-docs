Свойство AllowAddNode

См. также  Пример  Применяется к

Возвращает разрешение на добавление узла в дереве.

Свойство только для чтения.

 

Синтаксис

object.AllowAddNode

Синтаксис свойства AllowAddNode состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерево

 

Тип данных

Логический
