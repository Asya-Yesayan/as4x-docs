Свойство ECaption

См. также  Пример  Применяется к

Возвращает или устанавливает значение на иностранном языке заголовка дерева-справочника.

Свойство для чтения и записи.

 

Синтаксис

object.ECaption [=sValue] 

Синтаксис свойства ECaption состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта дерево
sValue строковое выражение, определяющее заголовок дерева на иностранном языке
