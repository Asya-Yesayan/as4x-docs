Свойство isSelected

См. также  Пример  Применяется к

Возвращает признак того, выбрана ли строка из произвольного вспомогательного списка.

Свойство для чтения.

 

Синтаксис

object.isSelected

Синтаксис свойства isSelected состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа произвольный вспомогательный список.

 

Тип данных

Логический
