Метод Show

См. также  Пример  Применяется к

Активизирует, выводя на экран, произвольный вспомогательный список.

 

Синтаксис

object.Show (ViewCode, Params())

Синтаксис метода Show состоит из следующих частей:

Параметр Описание
object строковое выражение, определяющее переменную, ссылающуюся на экземпляр объекта типа произвольный вспомогательный список.
ViewCode выражение типа Variant, определяющее идентификатор вида просмотра. Значение должно содержать либо идентификатор, либо объект типа вида просмотра.  
Params() массив параметров для подстановки в параметры вида просмотра.
