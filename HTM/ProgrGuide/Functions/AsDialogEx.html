Объект описательный диалог

Описательный диалог должен быть заранее описан и загружен в систему функцией LoadDialog.

Свойства Описание
AddAdditionalRegKey Возвращает или устанавливает дополнительное расширение для запоминания и загрузки значений из реестра.
Cancel Возвращает или устанавливает тип выхода из описательного диалога.
Caption Возвращает или устанавливает заголовок описательного диалога.
ChangeTypeControl Возвращает или устанавливает тип для данного элемента управления диалога.
CheckValue Возвращает или устанавливает признак проверки введенного значения на наличие в указанных папке или дереве.
ClickDropDownEvent Возвращает или устанавливает должно ли ядро системы генерировать событие ClickDropDown для данного элемента управления диалога.
Control Возвращает элемент управления описательного диалога.
ECaption Возвращает или устанавливает заголовок описательного диалога на иностранном языке.
ExistControl Определяется существование в описательном диалоге элемента управления с указанным идентификатором.
Name Возвращает идентификатор диалога.
Properties Возвращает временную коллекцию Dictionary для пользовательской обработки.
ReadOnly Возвращает или устанавливает статус режима редактирования заданного элемента управления.
Value Возвращает или устанавливает значение элемента управления диалога.

 

Методы Описание
SetFocus Передается фокус элементу управления с указанным идентификатором.
Show Активизирует, выводя на экран, описательный диалог.
