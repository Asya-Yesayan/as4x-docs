Свойство GetDialogParCount

См. также  Пример  Применяется к

Возвращает количество параметров диалога-филтра вида просмора.

 

Синтаксис

object.GetDialogParCount()

Синтаксис свойства GetDialogParCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта   AsBrowserParametersInfo
