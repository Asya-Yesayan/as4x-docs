Метод AddParam

См. также  Пример  Применяется к

Добавляет новый параметр в динамический источник данных.

Свойство для чтения.

 

Синтаксис

object.AddParam[sName, sDescription, sType, [vDefaultValue]]

Синтаксис метода AddParam состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, являющееся ссылкой на экземпляр объекта типа динамический источник данных.
sName обязательное строковое выражение, определяющее идентификатор добавляемого параметра.
 sDescription обязательное строковое выражение, определяющее описание добавляемого параметра.
sType обязательное строковое выражение, определяющее внутренний тип добавляемого параметра.
vDefaultValue необязательное выражение типа вариант, определяющее значение по умолчанию для добавляемого параметра.
