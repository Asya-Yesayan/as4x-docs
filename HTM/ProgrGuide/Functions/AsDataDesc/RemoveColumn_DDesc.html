Метод RemoveColumn

См. также  Пример  Применяется к

Удаляет колонку из данного вида динамического источника данных.

Свойство для чтения.

 

Синтаксис

object.RemoveColumn [sName]

Синтаксис метода RemoveColumn состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево.
sName обязательное строковое выражение, определяющее идентификатор удаляемой колонки.
