Метод GetParINdex

См. также  Пример  Применяется к

Возвращает индекс параметра по заданному имени.

Свойство для чтения.

 

Синтаксис

object.GetParINdex[sParName]

Синтаксис метода GetParINdex состоит из следующих частей:

Параметр Описание
object обязательное строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево.
sParName обязательное строковое выражение, определяющее наименование параметра, индекс которого возвращается.
