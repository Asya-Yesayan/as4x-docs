Свойство HeadLinesCount

 См. также   Пример  Применяется к

Возвращает или устанавливает ширину заголовков колонок.

Свойство для чтения.

 

Синтаксис

object.HeadLinesCount[=iValue] 

Синтаксис свойства HeadLinesCount состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта типа динамическое дерево
iValue численное выражение, определяющее ширину заголовков колонок динамического источника данных. По умолчанию установлено значение 2.
