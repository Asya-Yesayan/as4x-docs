Метод Remove

 См. также  Пример  Применяется к

Удаляет файловое дополнение из коллекции.

 

Синтаксис

object.Remove (sFileName)

Синтаксис метода Remove состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта коллекция файловых дополнений.
sFileName строковое выражение, определяющее наименование удаляемого файлового дополнения.
