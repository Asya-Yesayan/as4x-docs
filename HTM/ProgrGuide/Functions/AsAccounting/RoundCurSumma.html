Свойство RoundCurSumma

См. также  Пример  Применяется к

Возвращает степень округления валютной суммы для проводок данного учета. При сохранении проводок методом StoreFact, сумма проводки в валюте округляется до заданной позиции после десятичной запятой.


Свойство для чтения.

 

Синтаксис

object.RoundCurSumma

Синтаксис свойства RoundCurSumma состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на объект типа учет

 

Тип данных

Длинное целое
