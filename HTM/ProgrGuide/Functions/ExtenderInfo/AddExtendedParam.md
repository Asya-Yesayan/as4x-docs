# AddExtendedParam մեթոդ

[См. также](../ExtenderInfo.md) [Օրինակ](../ASDATA/OpenCursor.md#օրինակ) [Применяется к](../ExtenderInfo.md)

Սահմանում է տվյալների աղբյուրի ընդլայնման պարամետրի նկարագրությունը, որը կիրառվելու է տվյալների աղբյուրի հաշվարկելիս։

## Շարահյուսություն

```vb
Public Sub AddExtendedParam(ByVal sName As String, ByVal sType As String, ByVal vValue As Variant)
```

Բաղադրիչներն են՝

| Պարամետր | Նկարագրոթյոն |
|--|--|
| sName | Պարամետրի ներքին անունը։ |
| sType | Պարամետրի տվյալների տիպը։   |
| vValue | Պարամետրի արժեքը։ |
