Метод ClearParameters

См. также  Пример  Применяется к

Очищает список параметров для SQL запроса источника данных.

 

Синтаксис

object.ClearParameters([SetDefaultValues])

Синтаксис метода ClearParameters состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта источник данных.
SetDefaultValues необязательное выражение типа Boolean, определяющее значение параметров после очистки. Если SetDefaultValues = True, то параметры устанавливаются равными значению по умолчанию для типа, в противном случае параметры устанавливаются Empty. Знаение по умолчанию False.
