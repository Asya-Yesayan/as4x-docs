Метод Break

 См. также  Пример  Применяется к

В текущей строке печатной формы документа вставляет символ прогона страницы.

 

Синтаксис

object.Break

Синтаксис метода Break состоит из следующих частей:

Параметр Описание
object строковое выражение, ссылающееся на объект печатной формы.

 

Примечание

Данный метод вызывается из обработчика системного события Print.
