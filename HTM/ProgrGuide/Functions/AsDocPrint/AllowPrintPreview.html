Свойство AllowPrintPreview

См. также  Пример  Применяется к

Устанавливает разрешение на просмотр печатной формы документа из шаблона печати.

Свойство для записи.

 

Синтаксис

object.AllowPrintPreview[ = bValue ]

Синтаксис свойства AllowPrintPreview состоит из следующих частей:

Параметр Описание
object строковое выражение, ссылающееся на объект печатной формы.
bValue логическое выражение, разрешающее просмотр печатной формы документа.
