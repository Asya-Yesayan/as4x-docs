Свойство KeyList

 См. также  Пример  Применяется к

Возвращает или устанавливает список значений поисковой колонки для элементов коллекции строк списка. 

Свойство для чтения и записи.

 

Синтаксис

object.KeyList = [sKeyListValue]

Синтаксис свойства KeyList состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта коллекция строк списка.
sKeyListValue строковое выражение, определяющее список значений поисковой колонки для элементов коллекции. В списке значения перечисляются через запятую.

 

Тип данных

Строка
