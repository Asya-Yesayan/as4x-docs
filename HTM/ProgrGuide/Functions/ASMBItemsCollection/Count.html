Свойство Count

 См. также  Пример  Применяется к

Возвращает количество элементов в коллекции строк списка.

Свойство только для чтения.

 

Синтаксис

object.Count

Синтаксис свойства Count состоит из следующих частей:

Параметр Описание
object строковое выражение, являющееся ссылкой на экземпляр объекта коллекция строк списка.

 

Тип данных

Целое число
