# DOCLOG աղյուսակ

Այս աղյուսակը նախատեսված է փաստաթղթերի նկատմամբ կատարված գործողությունների պատմության գրանցման համար։

| Սյան անվանում | Տվյալների տիպ | Null      | Նկարագրություն |
|---------------|---------------|-----------|----------------|
| fISN          | int           | NOT NULL  | Փաստաթղթի ներքին նույնականացման համարը (isn): |
| fSUID         | smallint      | NOT NULL  | Գործողությունը գրանցող օգտագործողի ներքին համարը (կոդը)։ |
| fDATE         | datetime      | NOT NULL  | Գործողության կատարման ամսաթիվ/ժամանակը։ |
| fOP           | char (1)      | NOT NULL  | Փաստաթղթի նկատմամբ կատարված գործողության կոդը՝ <br> 'A' - Հասանելիության փոփոխություն <br> 'B' - Պատճենում <br> 'C' - Վիճակի փոփոխություն <br> 'D' - Հեռացում <br> 'E' - Խմբագրում <br> 'F' - Գրանցում որպես սևագիր <br> 'G' - Վավերացման ուղարկված փաստաթղթի մերժում <br> 'H' - Փաստաթղթի հեռացում՝ մեկ այլ փաստաթղթի հեռացման ընթացքում <br> 'I' - Ներմուծում <br> 'J' - Կցված ֆայլի հեռացում <br> 'K' - Կցված ֆայլի ավելացում <br> 'L' - Փաստաթղթի փոփոխում մեկ այլ փաստաթղթի կողմից <br> 'M' - Ծրագրային հաղորդագրություն, որը գրանցվում է Document.WriteLog մեթոդի միջոցով <br> 'N' - Ստեղծում <br> 'R' - Փաստաթղթի բոլոր հաշվառումների հեռացում HI, HI2 աղյուսակներից <br> 'S' - Փաստաթղթի կրկնակի մուտքագրում (StoreMode.SecondInput) <br> 'T' - Հառվառման գործառնություն(ներ)ի (Fact) գրանցում <br> 'U' - Ծրագրային մշակում (կիրառելի է միայն փաստաթղթի CheckLevel-ի WriteSoftwareLog(32) արժեքի դեպքում) <br> 'V' - Հաստատման ուղարկված փաստաթղթի մերժում  <br> 'W' - Հաստատման ուղարկված փաստաթղթի հաստատում <br> 'X' - Օգտագործողի կողմից նկարագրված ռեկվիզիտների (UDR) խմբագրում <br> 'Y' - Արխիվացում     |
| fSTATE         | smallint     | NOT NULL  | Փաստաթղթի վիճակը գործողության կատարման ընթացքում։ |
| fSUIDCOR       | smallint     | NULL      | Գործողությունը արդյունքը ստացող  օգտագործողի ներքին համարը (կոդը)։ |
| fCOM           | varchar(64)  | NULL      | Գործողությունը նկարագրող մեկնաբանություն։ |
| fCOMPNAME      | varchar(32)  | NULL      | Այն համակարգչի անունը, որից կատարվել է գործողությունը։ |
| fDCRID         | int          | NOT NULL      | Փաստաթղթի փոփոխման հայտի համարը։  |
| fAPICLIENTID   | smallint     | NOT NULL      | Այն կլիենտ ծրագրի ներքին նույնականացման համարը (id), որի շրջանակներում կատարվել է գործողությունը։ |
