Синтаксис

        Label {
                            Name = sLabelName;
                            Caption = sLabelCaption;  
                            ECaption = sLabelECaption;
                            Alignment = sLabelAlignment;
                            DisplayAtRight = nDisplayAtRight;
                            Distance = nDistance;
   
               };
  Параметр Описание
sLabelName строковое выражение, определяющее идентификатор метки
sLabelCaption строковое выражение, определяющее заголовок метки
sLabelECaption
 строковое выражение, определяющее заголовок метки на иностранном языке
sLabelAlignment необязательное строковое выражение, определяющее выравнивание метки относительно документа. При значении "LEFT" - выравнивание метки дается с лева от края формы документа, при значении "RIGHT" - выравнивание метки дается с права от края формы документа, а при значении "CENTER" - выравнивание метки дается по середине. По умолчанию принимает значение  "LEFT".
nDisplayAtRight необязательное численное выражение, принимающее значения 0 или 1. По умолчанию принимает значение 0, которое означает, что метка будет установлена с новой строки. При значении 1 метка будет установлена справа от предыдущего реквизита.
nDistance необязательное численное выражение, определяющее расстояние метки либо от левого края формы при  nDisplayAtRight = 0, либо справа от предыдущего реквизита  при nDisplayAtRight = 1.

 

Примечание

 При nDisplayAtRight = 1  значение sLabelAlignment игнорируется.
