Тип данных C( )

Символьный тип данных.

 

Синтаксис

C(nLen)

Синтаксис типа C() состоит из следующих частей:

Параметр Описание
nLen числовое выражение, определяющее количество символов. Максимально допустимое значение для колонок грид таблицы равно 255 символов, а в остальных случаях равно 512.

Пример использования символьного типа
